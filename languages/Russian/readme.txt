
Чтобы получить ВАШЕ-язычную версию MessageSearch 1.0, 
необходимо выполнить следующие шаги:

1. Все сообщения в файле <CONSTANTS_RUS.INC> должны быть переведены на ВАШ язык, 
   and name it in according to your language: <CONSTANTS_FRA.INC> or <CONSTANTS_GER.INC>
   This text file must be readable in Microsoft Word, using appropriate unicode font.
   
   To determine your local language codepage:
     Press <START> on Windows taskbar.
     <Run...>
     type: regedit
     <OK>
     locate key: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Nls\CodePage
     locate value on right side: ACP

   Possible values of msFontCharset constant is:
     ANSI_CHARSET
     SYMBOL_CHARSET
     SHIFTJIS_CHARSET
     HANGEUL_CHARSET
     GB2312_CHARSET
     CHINESEBIG5_CHARSET
     JOHAB_CHARSET
     HEBREW_CHARSET
     ARABIC_CHARSET
     GREEK_CHARSET
     TURKISH_CHARSET
     VIETNAMESE_CHARSET
     THAI_CHARSET
     EASTEUROPE_CHARSET
     RUSSIAN_CHARSET
     BALTIC_CHARSET

2. Carefully read <engl-Requirements.doc> and prepare source texts to be inserted into 
   your-language database (see example: <engl-example.txt>).
   
3. Translate User's Manual for program (<User's Guide.doc>) into YOUR language.

4. Send <CONSTANTS_xxx.INC>, <xxx-User's Guide.doc>, and several archived source text 
   files with sermons to email address: bigit@one.lv
   
   We compile YOUR-language version of program, and send it to you for test.
   When it will work correct, you can start serious work of preparing
   source text files for database.
